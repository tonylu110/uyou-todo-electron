## Run and build

- run in development mode
  ```bash
  pnpm i
  pnpm electron:serve // run in Windows
  pnpm electron:servemac // run in mac or Linux
  ```
- build
  ```bash
  pnpm i
  pnpm electron:build // build for Windows
  pnpm electron:buildmac // build for mac or Linux
  ```
